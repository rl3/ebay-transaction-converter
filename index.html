ebay Transaction Report Konverter

Hier können Sie die von ebay heruntergeladenen Transaction Reports im CSV-Format so umwandeln lassen, dass für alle Umsätze eine weitere Zeile mit den Gebühren eingefügt wird.

Alle Spalten in deren Titel Gebühr oder Verkaufsprovision vorkommen, werden als Summe in der folgenden Zeile in der Spalte mit Transaktionsbetrag zusammengefasst. Als Typ wird hier Andere Gebühr angegeben.

Bitte speichern Sie die CSV-Datei mit Semikolon oder Tabulator als Trenner! Andere Formate werden nicht unterstützt.

Die konvertierten Dateien werden wieder zum Speichern angeboten oder direkt in Ihrem Download-Ordner abgelegt.

Die Daten werden ausschließlich auf Ihrem Computer verarbeitet und nicht ins Internet gesendet.
